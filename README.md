## Description

API Nodejs 

## Installation Or ( Yarn )

```bash
$ npm install
$ OR
$ yarn install
```

## Running the app

```bash
# development
$ npm run dev
$ OR
$ yarn dev


# watch mode
$ npm run dev

# production mode
$ npm run start:prod
```

## Test

```bash

$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
